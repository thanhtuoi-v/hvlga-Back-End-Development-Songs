# Songs Backend API

This is a RESTful API for managing songs, built using Flask and MongoDB.

## Technologies

- Python 3.x
- Flask
- PyMongo
- MongoDB
- Pytest


## Config `env`
```
MONGODB_SERVICE=localhost:27017
MONGODB_USERNAME=
MONGODB_PASSWORD=
```

## Run app

```bash
python app.py
```

## Run test

```bash
pytest -v
```
